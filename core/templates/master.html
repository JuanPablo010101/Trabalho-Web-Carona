Sobre o Sistema de Carona

O Sistema de Carona IFMG foi desenvolvido para permitir que estudantes e funcionários do IFMG se cadastrem, ofereçam caronas ou procurem caronas para facilitar o deslocamento entre o campus e os arredores.

Este projeto foi desenvolvido com o intuito de proporcionar mais praticidade e segurança aos usuários, utilizando tecnologia web para facilitar o processo de encontrar e oferecer caronas de forma eficiente.
